# Images for Robotics Directory

This directory contains images for the Robotics Directory website.

## Structure

- `humanoid/` - Images of humanoid robots
- `robodog/` - Images of robot dogs
- `table/` - Images of table robots
- `entertainment/` - Images of entertainment robots
- `educational/` - Images of educational robots
- `household/` - Images of household robots
- `vacuum/` - Images of vacuum robots
- `logos/` - Logo images and branding assets

## Placeholder Images

- `placeholder.svg` - Default placeholder for robots without images
- `image-not-found.webp` - Fallback image when robot image fails to load

## Image Guidelines

- Use WebP format for optimal compression and quality
- Recommended dimensions: 400x400 pixels for robot cards
- Keep file sizes under 100KB for faster loading
- Include alt text descriptions in the data files for accessibility
